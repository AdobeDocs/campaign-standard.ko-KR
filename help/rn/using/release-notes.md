## 릴리스 26.2 {#26.2}

### 보안 수정 사항 {#26-2-security}

* 이번 릴리스에서는 보안 수정 사항이 적용됩니다.
* 이 릴리스는 다음과 같은 보안 업그레이드와 함께 제공됩니다. Debian 13 및 PostgreSQL 17.

### 기타 해결 사항 {#26-2-fixes}

* 동적 보고 기능을 개선하여 견고성을 높이고 데이터 불일치를 줄였습니다. (CAMP-58964)
* 안정성 향상을 위해 IMS 로그인 메커니즘이 개선되었습니다. (CAMP-59418)
* 페이지 매김된 요청에 대해 더 나은 URL 처리로 AEM 콘텐츠 통합이 개선되었습니다. (CAMP-59540)
* 옵션 화면에서 자동 해시를 사용하여 암호 유형 입력 지원을 추가했습니다. (CAMP-60206)
* 임시 워크플로우에서 잘못된 시퀀스 ID를 사용하던 문제를 수정했습니다. (CAMP-59824)
* AEM 캠페인 콘텐츠를 검색할 때 오류가 발생할 수 있는 문제를 해결했습니다. (CAMP-59926)
